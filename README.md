# 📊 Mutual Fund Overview & Insights

This project analyzes mutual fund data to identify **top-performing schemes with high returns and low risk** using Python, Excel, and Power BI.

🛠️ **Tools Used:** Python (Pandas, Sklearn), Excel, Power BI
📁 **Dataset:** 2500+ Mutual Fund Schemes (Top 30 Filtered)
---

## 🧠 Project Goal

To identify **top-performing, low-risk mutual fund schemes** using data-driven techniques and present actionable insights through an interactive Power BI dashboard.

---

## 🐍 Python-Based Fund Analysis

The analysis begins with exploring and processing a dataset of over 2500 mutual fund schemes.

### 🔹 Data Cleaning

* Removed irrelevant columns
* Handled missing values
* Standardized numeric formats (returns, expense ratios)

### 🔹 Data Analysis

* Generated statistical summaries (mean, median, std deviation)
* Analyzed distribution of returns, risk levels, and fund age

### 🔹 Data Normalization

* Applied `MinMaxScaler` for scaling features
* Enabled fair comparison across metrics

### 🔹 Fund Scoring & Ranking

Custom scoring based on:

* High 3-Year Returns
* Low Expense Ratio
* Optimal Fund Age
* Positive 1-Year Return

### 🔹 Final Output

* Selected **Top 30 Mutual Funds** balancing return and risk

---

## 📈 Power BI Dashboard – Insights

An interactive dashboard was created to visualize key insights and trends.

### 📌 Key Features

#### 📅 Dynamic Filters

#### 📊 Visual Insights
